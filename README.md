# 🌾 FarmSaathi

FarmSaathi is a smart and simple web application designed to help farmers track crop expenses, calculate profit/loss, and analyze performance easily.

---

## 🚀 Features

- 🌐 Bilingual Support (English & Kannada)
- 🎤 Voice Input for quick data entry
- 📊 Automatic Expense & Profit Calculation
- 🚨 Loss Alert (Warns if loss exceeds 10%)
- 🏆 Top Performing Crop Tracking
- 💾 Data stored using LocalStorage (no database required)

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  

---

## 📈 Formula Used

Profit = (Quantity × Rate) - Total Expense

---

## 🔧 How to Use

1. Enter village and crop details  
2. Add expense values (seeds, fertilizer, labour, irrigation)  
3. Enter market details (quantity & rate)  
4. Click **Add Record**  
5. View profit/loss instantly
